Maqueta de plumillas paso de tren

Implementamos un sistema de control para una maqueta de plumillas en un paso de tren, utilizando Arduino. El sistema detecta la proximidad del tren mediante sensores y activa las plumillas para garantizar la seguridad vial. Este proyecto combina electrónica y programación para simular un entorno ferroviario realista
